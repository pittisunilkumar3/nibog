# Event Image create

POST https://ai.alviongs.com/webhook/nibog/eventimage

Payload:-

{
  "event_id": 123,
  "image_url": "https://example.com/images/event123.jpg",
  "priority": 1,
  "is_active": true
}

Responsive:-

[
    {
        "id": 2,
        "event_id": 123,
        "image_url": "https://example.com/images/event123.jpg",
        "priority": 1,
        "is_active": true,
        "created_at": "2025-09-15T08:24:01.024Z",
        "updated_at": "2025-09-15T08:24:01.024Z"
    }
]

}
