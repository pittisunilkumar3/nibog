üìã Instructions for Manual Testing

  1. Open Registration Page: Navigate to /register-event
  2. Open Browser Console: Press F12 and go to Console tab
  3. Fill Form: Complete all required fields
  4. Select DOB: Click on "Child's Date of Birth" and select July 1, 2024
  5. Monitor Console: Look for these log messages:
    • DOB BUTTON CLICKED - When you click the date button
    • CALENDAR COMPONENT onSelect CALLED - When you select a date
    • DOB CHANGE HANDLER CALLED - When the date is processed
    • DOB TRACKING IN REGISTRATION - When creating booking data
    • CREATING PENDING BOOKING - When calling the API
  6. Check Network Tab: Monitor the pending booking API call payload
  7. Verify DOB: Ensure the payload contains "childDob": "2024-07-01"

üîç Common Issues to Check

  • SessionStorage Override: Clear browser storage before testing
  • Date Selection: Ensure you're clicking on July 1st, not July 31st
  • Calendar Component: Verify the selected date is highlighted correctly
  • Form State: Check that the DOB button shows the correct date after selection
  • API Payload: Verify the network request contains the correct DOB
